# Этап 7: Интерфейс (Веб-ориентированная траектория)

## Выполненные артефакты

| № | Артефакт | Статус | Расположение |
|---|----------|--------|--------------|
| 1 | Веб-страницы (5+) | ✅ | src/main/resources/templates/ |
| 2 | Контроллеры (Spring MVC) | ✅ | src/main/java/.../controller/ |
| 3 | DTO (Data Transfer Objects) | ✅ | src/main/java/.../dto/ |
| 4 | REST API (8+ эндпоинтов) | ✅ | src/main/java/.../controller/ |
| 5 | Адаптивная вёрстка | ✅ | src/main/resources/static/css/ |

## Список страниц

| Страница | URL | Описание |
|----------|-----|----------|
| Главная | / | Добро пожаловать |
| Список комнат | /rooms | Все переговорные комнаты |
| Календарь комнаты | /rooms/{id} | Расписание конкретной комнаты |
| Создание бронирования | /bookings/create | Форма бронирования |
| Мои бронирования | /bookings/my | Список бронирований пользователя |
| Вход | /login | Аутентификация |
| Регистрация | /register | Регистрация нового пользователя |
| Админ-панель | /admin | Управление (только для админа) |

## REST API эндпоинты

| Метод | URL | Описание |
|-------|-----|----------|
| POST | /api/auth/login | Аутентификация |
| POST | /api/auth/register | Регистрация |
| GET | /api/rooms | Список комнат |
| GET | /api/rooms/{id} | Комната по ID |
| GET | /api/rooms/available | Свободные комнаты |
| POST | /api/bookings | Создание бронирования |
| GET | /api/bookings/user/{userId} | Бронирования пользователя |
| DELETE | /api/bookings/{id} | Отмена бронирования |
