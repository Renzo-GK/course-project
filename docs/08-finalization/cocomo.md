## Расчет

### 1. Размер проекта
- Объем кода: **2000** строк
- Классы: 15+
- Методы: 50+

### 2. Коэффициенты (COCOMO Basic)
- **a = 2.4** (органический режим)
- **b = 1.05** (органический режим)
- **c = 2.5**
- **d = 0.38**

### 3. Трудоемкость (человеко-месяцы)
\\\
E = a * (KLOC)^b
E = 2.4 * (2.0)^1.05
E = 2.4 * 2.07
E = 4.97 человеко-месяцев
\\\

### 4. Длительность разработки (месяцы)
\\\
D = c * (E)^d
D = 2.5 * (4.97)^0.38
D = 2.5 * 1.83
D = 4.57 месяца
\\\

### 5. Количество разработчиков
\\\
N = E / D
N = 4.97 / 4.57
N = 1.09 ≈ 1 разработчик
\\\
