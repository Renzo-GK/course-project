# Диаграмма бизнес-контекста (IDEF0 A-0)

## Общая информация
**Название процесса:** Управление бронированием переговорных комнат

## Диаграмма
![Диаграмма IDEF0](images/context-diagram.png)

## Описание элементов

### ВХОД (Что поступает в систему)
| Элемент | Описание |
|---------|----------|
| Данные пользователя | Логин, пароль, ФИО, роль |
| Данные о комнате | Название, вместимость, оборудование |
| Данные о бронировании | Дата, время начала и окончания |

### ВЫХОД (Что производит система)
| Элемент | Описание |
|---------|----------|
| Расписание | Отображение занятости всех комнат |
| Подтверждение | Успешное/неуспешное бронирование |
| Отчеты | Статистика по использованию комнат |

### УПРАВЛЕНИЕ (Правила и ограничения)
- Уникальность бронирования
- Валидация времени
- Минимальная длительность - 15 минут
- Права доступа: USER и ADMIN

### МЕХАНИЗМЫ (Исполнители)
- Веб-браузер
- Spring Boot
- PostgreSQL
