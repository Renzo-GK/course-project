# Веб-сервис для бронирования переговорных комнат

## О проекте
Веб-приложение для управления бронированием переговорных комнат и рабочими пространствами в офисе.

## Траектория
**Б. Веб-ориентированная**

## Технологический стек
- **Backend:** Java 17, Spring Boot, Spring Data JPA, Spring Security
- **Frontend:** Thymeleaf, HTML, CSS, JavaScript
- **База данных:** PostgreSQL
- **Сборка:** Maven
- **Контроль версий:** Git, GitHub
